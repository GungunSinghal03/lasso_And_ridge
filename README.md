# 🏡 Lasso & Ridge Regression on California Housing Dataset

This project demonstrates the implementation of **Lasso Regression** and **Ridge Regression** using the California Housing dataset. It focuses on understanding how regularization improves model performance and prevents overfitting.

---

## 📌 Project Overview

- Dataset: California Housing Dataset
- Goal: Predict house prices using regression techniques
- Algorithms Used:
  - Linear Regression (baseline)
  - Ridge Regression (L2 Regularization)
  - Lasso Regression (L1 Regularization)

---

## ⚙️ Tech Stack

- Python
- NumPy
- Pandas
- Scikit-learn
- Matplotlib (optional for visualization)

---

## 📊 Steps Performed

1. Data Loading
2. Data Preprocessing
   - Handling features
   - Feature scaling using StandardScaler
3. Train-Test Split
4. Model Training
   - Linear Regression
   - Ridge Regression
   - Lasso Regression
5. Model Evaluation
   - R² Score
   - Mean Squared Error

---

## 📈 Results

- Compared performance of Lasso and Ridge models
- Observed impact of regularization on overfitting
- Identified optimal alpha values for better accuracy
